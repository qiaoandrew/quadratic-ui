## Examples

### Sort Order
To set sort order, add the `sortOrder` prop as `ascending` or `descending`. Set it to `none` to disable sorting.
<ComponentPreview id="bar-list-sort-order" />

### Value Formatter
Use the `valueFormatter` prop to format the value of each item in the list.
<ComponentPreview id="bar-list-value-formatter" />

### Link
When `href` is in the array, labels become anchor tags.
<ComponentPreview id="bar-list-external-link" />

### Clickable
When `onValueChange` is added to the `BarList`, the bars become clickable.
<ComponentPreview id="bar-list-clickable" />

### Link and Clickable
`href` and `onValueChange` can be combined.
