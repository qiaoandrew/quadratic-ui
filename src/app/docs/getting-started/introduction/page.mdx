[shadcn/ui](https://ui.shadcn.com/) is amazing. It provides developers with an easy way to add components to their apps and makes customization really easy.

I've been using shadcn/ui for the better part of a year now, and I often found myself making the same little tweaks when creating new projects. So, I decided to create a place to store these edited components.

At the same time, I also found out that a lot of my friends were very skilled engineers building great projects, but didn't have the interest in developing interfaces to showcase their creations. Thus, I decided to create a documentation website around my components.

That said, quadratic/ui is basically a slightly edited version of shadcn/ui, complete with a 1:1 Figma file (which took me hours upon hours to make 😭).

I really hope it helps and you have fun building with it! Please feel free to contribute!

If you would like to get in touch, feel free to send me a message on [LinkedIn](https://www.linkedin.com/in/andrewqiao/) or [X/Twitter](https://twitter.com/andrewwqiao).
