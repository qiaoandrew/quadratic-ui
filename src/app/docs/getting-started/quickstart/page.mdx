<Title>Quickstart</Title>
<Description>Add quadratic/ui to your Next.js app.</Description>

<Steps>

<StepTitle>Create a Next.js app with TypeScript and Tailwind CSS. I recommend using [create-t3-app](https://create.t3.gg/en/installation#pnpm) with [pnpm](https://pnpm.io/). Check out my [detailed guide](/docs/getting-started/create-t3-app) on create-t3-app. Otherwise, you can follow the [Next.js installation instructions](https://nextjs.org/docs/app/api-reference/create-next-app).</StepTitle>

<StepTitle>Add the following dependencies to your project.</StepTitle>
<StepContent>
```bash
pnpm add tailwindcss-animate class-variance-authority clsx tailwind-merge
```
</StepContent>

<StepTitle>Add the [Lucide](https://lucide.dev/) icon library to your project.</StepTitle>
<StepContent>
```bash
pnpm add lucide-react
```
</StepContent>

<StepTitle>Make the following changes to your `tailwind.config.ts` file.</StepTitle>
<StepContent>
Add the `darkMode` property to the root of your config.
```ts
export default {
  darkMode: ["class"],
  ...
} satisfies Config;
```

```ts
export default {
  ...
  plugins: [require("tailwindcss-animate")],
} satisfies Config;
```
