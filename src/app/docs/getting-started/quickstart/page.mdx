<Title>Quickstart</Title>
<Description>Add quadratic/ui to your Next.js 15 + React 19 app.</Description>

<Steps>

<StepTitle>Create a Next.js 15 + React 19 app any way you like. Our documentation uses pnpm, TypeScript, and Tailwind CSS, so those technologies are recommended. If you're starting a new project, follow my [Create T3 App guide](/docs/getting-started/create-t3-app), which provides the best starting point for building Next.js apps.</StepTitle>

<StepTitle>In your project root, run the following script to install some dependencies to your project.</StepTitle>
<StepContent>
```bash
pnpm add tailwindcss-animate tailwind-variants clsx tailwind-merge lucide-react
```
</StepContent>
