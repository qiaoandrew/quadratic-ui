# quadratic/ui

Component library for Next 15 + React 19 built using [shadcn/ui](https://ui.shadcn.com/) and [Radix UI](https://www.radix-ui.com/).

![hero](/public/og.jpg)

## Documentation

Visit https://quadratic-ui.com to install and start using components.
